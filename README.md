## **什么是 ChatGPT 镜像网站？**

**ChatGPT 镜像网站**（Mirror Site）是一个为满足特定用户使用需求而搭建的访问站点，通常将原始服务同步到国内服务器，优化访问速度，提升网络连接性能。
